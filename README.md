# Tower-Defense-Test
Testando criar um jogo simples de tower defense

# Descriçao
Esse jogo de Tower Defense foi feito para mostrar diferentes tipos de mecânicas para diferentes torres, que os jogadores podem utilizar para derrotar horas de inimigos.

# Características
- Diferentes tipos de torres com habilidades únicas
- Diferentes tipos de inimigos com características distintas
- Suporte a toque e integração com mobile
- UI e controles responsivos
- Sistema de melhorias para torres
- Prévia de alcance de torres antes de serem posicionadas
